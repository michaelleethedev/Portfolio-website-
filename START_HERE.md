## 🆘 Troubleshooting

### Site Won't Start
```bash
npm install
npm run dev
```

### Changes Don't Appear
- Save the file
- Check browser (should auto-reload)
- Clear cache if needed

### Resume Won't Download
- Ensure `resume.pdf` is in `public/` folder
- Check filename is exactly `resume.pdf`

### Links Don't Work
- Ensure URLs include `https://`
- Test in incognito mode
