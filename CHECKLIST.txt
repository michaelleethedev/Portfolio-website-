TROUBLESHOOTING
================================================================================

Site won't start?
  → Run: npm install
  → Then: npm run dev

Changes don't appear?
  → Save the file
  → Browser should auto-reload
  → Try clearing cache

Resume won't download?
  → Check file is named exactly "resume.pdf"
  → Check it's in the "public" folder

Links don't work?
  → Make sure URLs include https://
  → Test in incognito mode

Mobile looks broken?
  → Clear browser cache
  → Test in incognito mode
  → Check browser console for errors
